# WatchCatalyst - Real-time Trading News Intelligence Platform

A professional news intelligence platform that aggregates and analyzes breaking news from multiple sources to provide actionable trading signals for crypto, stocks, and derivatives markets.

## Features

### Real-time News Dashboard
- Live news feed with automatic updates
- Toggle-able auto-refresh with visual indicators
- Professional trading platform aesthetic (dark theme inspired by Bloomberg Terminal)
- Responsive design optimized for desktop and mobile

### Multi-Category News Aggregation
- **Crypto**: Bitcoin, Ethereum, altcoins, DeFi, memecoins
- **Stocks**: Market indices, tech stocks, commodities
- **Geopolitics**: War, conflicts, international relations affecting markets
- **Technology**: AI, blockchain, breakthrough innovations
- **Politics**: Elections, policy changes, regulatory developments
- **Animals**: Rare discoveries, biotech opportunities
- **Sports**: Major events with betting/trading implications

### AI-Powered Analysis
- **Sentiment Analysis**: Bullish, bearish, or neutral classification
- **Relevance Scoring**: 0-100 scale for trading actionability
- **Trading Signals**: Specific buy/sell/watch recommendations
- **Deep Insights**: Key takeaways for traders
- **Affected Assets**: Specific tokens, stocks, and tickers
- **Timeframe Analysis**: Immediate, short-term, medium-term, or long-term impact

### Advanced Search & Filtering
- Real-time search across all news articles
- Category-based filtering
- Keyword matching on titles and summaries

### Smart Notifications
- Toast notifications for new articles
- Visual indicators for live updates
- Non-intrusive notification system

## Technology Stack

- Next.js 16 with React 19
- TypeScript for full type safety
- Tailwind CSS v4
- Modern UI components

## Getting Started

### Installation

```bash
npm install
```

### Running the App

```bash
# Development mode
npm run dev

# Production build
npm run build
npm start
```

Visit `http://localhost:3000` to see the app.
